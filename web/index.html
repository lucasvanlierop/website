I'm a software engineer specialized in creating, refactoring or containerizing (web) applications. I'm also organizer of the DomCode conference and DomCode meetup.

Let me help you with:

  • Build a modern (web) application.

    I know my around in modern PHP and can help you by building a web application, API, microservice, middleware. with proper documentation, tests.

  • Refactor/improve legacy applications

    Many software projects turn into legacy projects over time. Reasons being the use of older, maybe even obsolete technologies. Lack of tests, close coupling. When the application is still (very) valuable to your company I can help to modernize the application, set up a testing strategy, write documentation.

  • Containerize (web) applications

    Many issues with applications are caused by differences in the dev/test/production pipeline. This can be prevented by making sure the stack of Operating system + web server + runtime + the application itself is consistent in every stage. Even more than with Virtual machines this can be achieved by using Linux containers.

  • Train and coach junior developers

    Junior developers are often very eager to learn and grasp knowledge. I can help them learning, making the right choices...

  • et up continuous integration / continuous deployment

    The era of waiting hours, days or even months before deploying a new version of an application to production are over. Current technologie always us to deploy fast and often. I can help preparing your application, setting up a CI server, setting up automatic deployment.

  • Reviewing/audit software

    If your owner of an application and you're not sure about it's state I can do a review/audit and come up with warn about areas that need attention, suggest improvments.
